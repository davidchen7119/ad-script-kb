# ad-script-kb

广告/宣传片解说词知识库 Skill —— 针对 WorkBuddy 的本地知识库管理能力扩展。

## 功能

自动化广告宣传片解说词的**下载 → 提取 → 转写 → 深度分析 → 归档**全流程。

- 🎬 支持新片场链接直接下载（yt-dlp）
- 🗣️ OpenAI Whisper 中文语音转写 + AI 人工校正
- 📖 导演视角深度分析（风格/结构/主题/脉络/视觉/风险）
- 💡 提取 8–12 条可复用创意元素
- 🌐 3 行业 × 3 主题跨界拓展
- 📂 本地 Markdown 知识库自动归档与索引

## 知识库结构

```
文案知识库/
├── inbox/              ← 用户丢新文件的地方
├── archive/            ← 处理完的文件按分类存放
│   ├── 政务城市/
│   ├── 企业商业/
│   ├── 文旅人文/
│   ├── 科教医疗公益/
│   └── 小众特色/
├── index.md            ← 索引导航（AI 维护）
├── templates/
│   └── style_reference.md  ← 11种解说词风格模板库
└── .workbuddy/memory/
```

## 风格分类体系（11 种）

| 类别 | 风格 |
|------|------|
| 政务城市类 | 恢弘大气风、民生纪实温情风 |
| 企业商业类 | 高端商务风、工业匠心风、年轻活力风 |
| 文旅人文类 | 诗意文艺风、历史厚重风 |
| 科教医疗公益类 | 治愈走心风、热血励志风 |
| 小众特色风格 | 极简理性风、生活化口语风 |

## 使用方式

在 WorkBuddy 中加载此 skill 后，发送视频链接或上传视频文件到 inbox 即可自动触发全流程。
